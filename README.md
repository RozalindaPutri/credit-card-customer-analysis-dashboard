# credit-card-customer-analysis-dashboard

## Project Overview
This project presents an interactive credit card customer analysis dashboard built using Google Looker Studio. The dashboard analyzes customer demographics, transaction behavior, revenue performance, and payment risk to support data-driven decision-making in the financial industry.

## Key Metrics
- Total Customers: 5,054  
- Total Transaction Volume: 326,544  
- Total Transaction Amount: $22.31M  
- Total Delinquent Accounts: 317  
- Interest Earned: $3.19M  
- Average Customer Satisfaction: 2.25
